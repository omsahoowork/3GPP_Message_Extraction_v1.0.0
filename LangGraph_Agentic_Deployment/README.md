# LangGraph Agentic Deployment

Streamlit GUI wrapper for the complete pipeline in `../LangGraph_Agentic_App`.

## What this app does

- Requires `Test Description` and `RAT (NR/LTE)` for every query.
- Runs full LangGraph pipeline end to end.
- Handles all graph interrupts with GUI option selection.
- Shows step analysis state and final outputs.
- Captures feedback (`thumbs_up` / `thumbs_down` + short text).
- Tries Google Sheets write first, then stores fallback entries in `feedback/feedback_fallback.jsonl`.

## Local run

1. Activate your environment.
2. Install deployment dependencies:

```bash
pip install -r requirements.txt
```

3. Run Streamlit:

```bash
streamlit run app.py
```

## Streamlit Community Cloud deployment

1. Push this repository to GitHub.
2. In Streamlit Community Cloud, create a new app.
3. Set app path to:

`KG_Generation_Pipeline/RAG_KG_Integration/LangGraph_Agentic_Deployment/app.py`

4. Set Python dependencies path automatically from this folder `requirements.txt`.
5. Add secrets in app settings:

- `GOOGLE_SHEET_ID`
- `GOOGLE_SHEET_WORKSHEET` (optional; default `Feedback`)
- `google_service_account` block from `.streamlit/secrets.toml.example`
- LLM and tracing variables as needed (`OPENAI_API_KEY`, `LANGCHAIN_*`)

## Notes

- The app writes runtime query configs under `runtime_configs/`.
- Sequence diagram and CSV outputs are generated by the underlying pipeline in `../LangGraph_Agentic_App/Evaluation/`.
- Free-tier environments may have timeout constraints for long extraction runs.
